About

This app parses and evaluates lambda calculus expressions using different β-reduction strategies.

How to use

  1. Enter a lambda expression (example: \x. x).
  2. Select a reduction strategy.
  3. Press Evaluate to normalize.

Syntax

  • Lambda: \x. body
  • Application: f x (use parentheses as needed)
  • Variables: x, foo, etc.

β-reduction strategies

  • Normal order: leftmost outermost first. If a normal form exists, normal order will find it.
  • Call-by-name (CBN): like normal order, but does not reduce inside lambdas.
  • Applicative order: rightmost innermost first (reduces arguments before applying). Can diverge even when a normal form exists.
